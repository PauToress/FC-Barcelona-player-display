## 🎯 项目简介

**Més que un club** —— *不仅仅是一家俱乐部*。

本项目以巴塞罗那足球俱乐部官方刊的视觉语言为基调，呈现：

- 23 名球员（11 首发 + 12 替补）的完整资料
- 4-3-3 阵型排布与球员位置高亮
- 5 大荣誉赛事（西甲、欧冠、国王杯、超级杯、世俱杯）的历史与夺冠明细
- 球队完整赛季赛程（已完赛 / 即将进行 / 直播中）
- 球员详情弹窗（个人资料、荣誉、转会记录、德转身价走势）

**目标用户**：巴萨球迷、体育媒体从业者、转会市场研究者、设计 / 前端学习者。

## ✨ 核心特性

| 模块 | 功能描述 |
|---|---|
| **多语言 UI** | 顶部"双语 / EN / 中文"一键切换，所有内容智能去重，官方名不重复翻译 |
| **球员详情弹窗** | 5 个 Tab（个人资料 / 荣誉 / 转会 / 德转身价 / 数据），高对比度文字 |
| **4-3-3 阵型图** | 球场 SVG 背景，11 名球员按阵型点位排布，按位置（门将 / 后卫 / 中场 / 前锋）高亮 |
| **队徽系统** | 50+ 支球队内嵌 SVG 队徽，FC Barcelona 标志性 Blaugrana 红蓝分裂 |
| **赛程卡片** | 双日期显示（左侧大字号 + 右上 DD/MM/YYYY），赛事 / 时间 / KO 居中突出 |
| **荣誉板块** | 每项荣誉独立区块，5 大要素：双语赛事科普、夺冠时间、主力阵容、进球功臣、合影位 |
| **响应式设计** | 桌面 / 平板 / 手机 3 档断点，移动端单列压缩 |
| **零后端** | 所有数据内嵌 `app.js`，无网络请求（除可选字体外），CDN 不稳定也不影响使用 |

## 🛠 技术栈

- **HTML5** — 语义化结构
- **CSS3** — 变量、Grid、Flexbox、自定义动画、媒体查询
- **原生 JavaScript (ES6+)** — 零依赖，无构建工具
- **Inline SVG** — 队徽 / 阵型图 / 身价折线图

**不使用** React / Vue / jQuery / Webpack / 任何 npm 依赖。

## 🚀 快速开始

### 方式 1：直接打开（最简单）

双击 `index.html` 即可在默认浏览器打开。

### 方式 2：本地服务器（推荐，避开部分浏览器跨域限制）

```bash
# Python 3
py -m http.server 8788 --bind 127.0.0.1

# 或 Node.js
npx http-server -p 8788

# 或 PHP
php -S 127.0.0.1:8788
```

然后浏览器访问：

```
http://localhost:8788/index.html
```

### 方式 3：VS Code Live Server

安装 `Live Server` 扩展 → 右键 `index.html` → **Open with Live Server**

## 🌐 多语言支持

顶部导航栏右上角有语言切换器：

| 按钮 | 模式 | 说明 |
|---|---|---|
| **双语** | bi | 中文 + 英文同步展示 |
| **EN** | en | 仅英文 |
| **中文** | cn | 仅中文 |

**智能去重**：官方名（巴塞罗那俱乐部 / Fútbol Club Barcelona）双语相同时**只展示一份**，避免"· 巴塞罗那俱乐部 · 巴塞罗那俱乐部"的重复。

所有 i18n 翻译集中在 `app.js` 顶部的 `T{}` 对象中，新增 / 修改翻译不需要改动 HTML。

## 🎨 设计系统

### 色彩（CSS 变量）

| 变量 | 颜色 | 用途 |
|---|---|---|
| `--garnet` | `#A50044` | 巴萨红，主按钮、强调色 |
| `--blau-deep` | `#0A2540` | 巴萨深蓝，标题、文本 |
| `--blau` | `#004D98` | 巴萨蓝，渐变 / 装饰 |
| `--gold` | `#C9A961` | 冠军金，分隔线 / 奖杯 |
| `--paper` | `#F5EFE6` | 米白背景 |
| `--ink` | `#1A1A1A` | 主文本 |
| `--ink-soft` | `#666` | 副文本 |

### 字体栈

| 用途 | 字体 |
|---|---|
| 中文标题 | `"PingFang SC", "Microsoft YaHei", "Source Han Sans CN", sans-serif` |
| 西班牙文标题 | `"Anton", "Bebas Neue", "Impact", sans-serif` |
| 正文英文 | `"DM Sans", "Inter", system-ui, sans-serif` |
| 正文中文 | `"Noto Sans SC", "PingFang SC", "Microsoft YaHei", sans-serif` |
| 数字 / 编号 | `"Oswald", "Bebas Neue", monospace` |

### 排版节奏

- Hero 标题：clamp(48px, 7vw, 112px) 中文 + clamp(22px, 2.4vw, 36px) 西文
- 模块大数字：clamp(80px, 10vw, 160px) 装饰编号
- 赛程日期：36px / 800 字重 / 深蓝

### 动画

- 卡片悬停：右移 4px + 主题色高亮
- 队徽悬停：1.12 倍放大 + 阴影增强
- 模态框弹出：scale + opacity 300ms 过渡
- 赛程卡片：IntersectionObserver 错峰淡入（800ms 安全网）

## 📊 数据来源与说明

- 球员基本资料：公开网络资料 + 编辑整理
- 荣誉数据：维基百科 / 百度百科 / 官方资料（已在 PRD 中详细列举）
- 队徽配色：基于俱乐部官方色彩还原（`#A50044` Garnet / `#0A2540` Blau-deep / `#C9A961` Gold）
- 队徽图形：**完全自绘内嵌 SVG**，无外链、无 404、无版权风险

> ⚠️ 本项目为**展示 / 学习用途**，所有球队标识、球员照片、荣誉数据仅作编辑参考，不用于任何商业目的。
